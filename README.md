# Porté Fashionshow
Minimal single page website for the [Porté](https://xn--port-epa.ch/) fashion show using [fullPage.js](https://alvarotrigo.com/fullPage/), [gulp.js](https://gulpjs.com/) and [Bower](https://bower.io/) 

## Installation
```
> npm install
> bower install
```

## Build
```
> gulp
```

## Preview
Check out `/public` folder
